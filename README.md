# Langchain Chatbot
This is a project for study purposes only. We want to build a smart chatbot using Langchain and GPT which can engage in natural conversations with user and also can look up for more information to generate better answers. 

We ultilize Agent module with tools provided by Langchain and GPT-3.5 chat model for our smart chatbot. 

The project is divided into frontend and backend, please review both repos for more information.
